# Lux & Lucid NYCS
Sound triggered 3D Model. Just a proof of concept.

## Installation
You need node and npm installed on your machine. It should work on all platforms electron supports. I only tested in on a linux machine.

```
npm install
```

## Run

```
npm start
```
